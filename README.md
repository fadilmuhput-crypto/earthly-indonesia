# Earthly Indonesia

**Bumi Hijau, Indonesia Maju**

Platform aksi lingkungan untuk membantu masyarakat Indonesia mengadopsi kebiasaan ramah lingkungan dan menciptakan gerakan nasional mengurangi dampak lingkungan.

## Fitur

- **Earth Score Assessment** — Kuis interaktif untuk mengukur skor keberlanjutan (0–100)
- **Green Action Library** — 50+ aksi ramah lingkungan di 5 kategori
- **Daily & Weekly Challenges** — Tantangan dengan poin dan lencana
- **Impact Dashboard** — Lacak CO₂, plastik, air, dan pohon setara
- **Community Campaigns** — Bergabung dengan kampanye dan leaderboard
- **Learning Hub** — Konten edukasi tentang lingkungan

## Tech Stack

Next.js 16 · TypeScript · Tailwind CSS v4 · shadcn/ui · Supabase · PostgreSQL · Vercel

## Mulai

```bash
npm install
cp .env.example .env.local
# Isi .env.local dengan kredensial Supabase Anda
npm run dev
```

Lihat [DEPLOYMENT.md](./DEPLOYMENT.md) untuk panduan deployment lengkap.
